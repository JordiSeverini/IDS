# Python Intrusion Detection System (IDS)

## Overview
A real-time network-based Intrusion Detection System (IDS) built in Python.  
Currently capable of capturing live network packets and extracting basic metadata.

## Features (Work-in-Progress)
- Capture live packets using **Scapy**
- Extract IP source, destination, and protocol
- Designed for modular expansion (future detection logic)

## Project Status
Early development / work-in-progress

Planned features:
- Ask user to select interface dynamically
- JSON-style logging of packet metadata
- Signature-based detection modules
- Behavioral anomaly detection
- Real-time alerting

## Technologies
- Python (scripting and modular design)
- Scapy (packet sniffing and network analysis)
- TCP/IP networking concepts
- JSON (planned for logging)
- Real-time packet processing

- Capturing and formatting packet metadata
